# Synexa Growth OS

An AI Executive Leadership Team for Every Business.

## 5D Framework
1. **Discover**: Business onboarding
2. **Design**: Business strategy generation
3. **Deliver**: Campaign and sales execution recommendations
4. **Develop**: Measure KPIs and learn from results
5. **Dominate**: Executive dashboard with business intelligence

## Project Structure
- `/frontend`: Next.js 15, React 19, Tailwind CSS, shadcn/ui
- `/backend`: Python, FastAPI, LangGraph
- `/shared`: Shared schemas, types, or protocols
- `/database`: Supabase migrations, seed data, and ChromaDB setups
- `/deployment`: Vercel (frontend) and Render (backend) configs, Dockerfiles
- `/scripts`: Utility scripts for development
- `/docs`: Architecture and API documentation
